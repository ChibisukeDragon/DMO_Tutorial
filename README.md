## 1. 简介

**DMO_Tutorial**是一个为入坑DMO玩家准备的快速上手指南。

在这里你可以快速了解游戏的机制、变强的路线、数码兽的知识等。但是还没写完，见谅。

> 注：因为在补充子内容的过程中，存在框架增添，和图片重复的情况。因此本仓的更新频率会降低，尽量每次上传时都补充比以往更新更多的内容。
